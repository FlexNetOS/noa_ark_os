### Best Practices

2. **Monitor confidence**: Track AI confidence over time

4. **Archive cleanup**: Follow retention policies

### Common Workflows

**Integrate External Library:**
```bash
cd crc/drop-in/incoming/
git clone https://github.com/external/lib.git
```

**Update Stale Code:**
```bash

```

**Query Archives:**
```bash
crc search "function_name" --in-archives
```
