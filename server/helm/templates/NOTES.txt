The NOA Unified Server has been deployed.

To monitor rollout status:
  kubectl get pods -n {{ .Release.Namespace }} -l app.kubernetes.io/instance={{ .Release.Name }}

HTTP endpoint:
  kubectl port-forward svc/{{ include "noa-server.fullname" . }} 8080:{{ .Values.service.port }}
  curl http://localhost:8080/health

Metrics endpoint:
  kubectl port-forward svc/{{ include "noa-server.fullname" . }} 9090:{{ .Values.service.metricsPort }}
  curl http://localhost:9090/metrics
