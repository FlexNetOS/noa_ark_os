## Available Agents

- **Policy Enforcer Commander Chief** - Ensures that all requests comply with
  security and usage policies before, during, and after being processed.
  Orchestrates subordinate policy agents.

- **DE-Dupe file Agent**
- Prevents duplicate file uploads by checking existing records. Ensures:
  efficient storage and retrieval, compliance with data management policies.
  file is routed through rtt (Relay Terminal Tool) for routing single file to
  multiple destinations without duplicating file or using symlinks.

- **Compliance Auditor Agent** - Audits actions and logs to ensure adherence to
  established policies.

- **Usage Monitor Agent** - Monitors system usage to detect and prevent policy
  violations.

- **Security Validator Agent** - Validates requests against security policies to
  prevent unauthorized access.
