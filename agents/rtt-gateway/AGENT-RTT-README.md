# AGENT-RTT-README.md

Describes the RTT Gateway agents available in this directory. RTT (Relay
Terminal Tool) Smart Hub that connects everything.

## RTT Gateway Agents

This directory contains a collection of agents specifically designed to
facilitate all connections through the RTT Gateway. These agents handle tasks
such as routing requests, managing sessions, and ensuring secure communication
between clients and backend services.

### Available Agents

- **RTT Router Agent** - Routes incoming requests to the appropriate backend
  services based on predefined rules and criteria.

### - Policy Enforcement Agent\*\* - Ensures that all requests comply with

security and usage policies before being processed.
