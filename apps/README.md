# Applications

Built-in system and user applications.

## System Applications

- Terminal/Shell
- File Manager
- System Monitor
- Configuration Manager
- Development Environment

## Application Framework

Provides APIs for building applications that run on NOA ARK OS.

## Structure

```
apps/
├── system/         # System applications
├── framework/      # Application framework
└── examples/       # Example applications
```

## Development

Applications can be written in any of the supported languages:
- Rust (for performance-critical apps)
- Python (for rapid development)
- Go (for concurrent services)
- C# (for GUI applications)
